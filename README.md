# express-basic-api

Express.js kullanılarak geliştirilmiş basit bir API.

## Kurulum ve çalıştırma

#### Projeyi klonlayın

```shell
git clone GITHUB_URL
```

#### Proje klasörüne girin

```shell
cd express-basic-api
```

#### Bağımlılıklar Yükleyin

```shell
npm install
```

#### Projeyi çalıştırın

```shell
npm start
```
